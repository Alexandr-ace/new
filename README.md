# 🕷️ Wildberries Parser

Парсер для сбора данных о товарах с Wildberries через внутренний API.

## 🛠️ Стек технологий
- Python 3.12
- Библиотеки: `curl-cffi`, `psycopg2-binary`
- База данных: PostgreSQL (в Docker)
- Инструменты: Docker, Docker Compose
